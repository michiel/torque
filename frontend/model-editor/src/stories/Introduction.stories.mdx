# Torque Model Editor - Storybook

Welcome to the Torque Model Editor component library! This Storybook contains all the UI components used in the Torque Model Editor application.

## What is Torque Model Editor?

The Torque Model Editor is a visual interface for creating and editing Torque application models. It provides a user-friendly way to:

- Create and manage data models
- Define entities and their relationships
- Configure validation rules and constraints
- Design UI layouts and forms
- Test and preview your models

## Components

### Navigation Components
- **Navigation**: The main sidebar navigation with search functionality
- **ConnectionStatus**: Shows real-time connection status to the backend

### UI Components
- **Button**: Standard button components with various styles and sizes
- **Forms**: Input fields, selects, and form validation components
- **Modals**: Dialog boxes for creating and editing entities

## Development

This Storybook is automatically generated from the components in the `src/components` directory. To add new stories:

1. Create a new `.stories.tsx` file in the `src/stories` directory
2. Follow the existing patterns for Meta configuration
3. Add decorators for MantineProvider and other contexts as needed
4. Define your story variants with different props

## Technology Stack

- **React**: UI framework
- **TypeScript**: Type safety
- **Mantine**: Component library
- **Storybook**: Component development environment
- **Vite**: Build tool
- **GraphQL**: Data fetching with Apollo Client
